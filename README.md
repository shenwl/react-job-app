# 功能

  - boss发布招聘信息，并且可以看到所有求职信息

  - 求职者可发布求职信息，并且可以看到招聘信息

  - 可以通过点击信息进行即时通讯

# 技术选型

- react + react-router + redux

- nodejs/express

- mongoose/mongoDB

# 使用

- 下载并安装mongoDB， 打开

- clone项目
  ```bash
  git clone git@github.com:shenwl/react-job-app.git
  cd react-job-app
  npm install
  ```

- 运行本地服务器，开始项目
  ```bash
  nodemon server/server.js
  npm start
  ```
